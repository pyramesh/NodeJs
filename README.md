# NodeJs
NodeJs integration with MongoDB

Technologies used

1.	NodeJs
2.	MongoDB
3.	Third Party Payment Gateway for Payments (sendGrid)
4.	Third Party Email providers (stripe)

Functionalities implemented

1.	User Registration
2.	DB authentication
3.	CRUD operations on Products 
4.	Shop 
5.	Cart
6.	Checkout
7.	Order
8.	Payment
